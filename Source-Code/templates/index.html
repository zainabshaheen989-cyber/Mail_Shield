This platform uses AI to scan emails and identify potential threats such as phishing, spam, and malicious links. A multi-layer detection system ensures reliable results with clear explanations.

Core Features
🧠

ML Model

TF-IDF vectorized Logistic Regression trained on 10,000+ labeled email samples for high-accuracy classification.

🔗

Link Analyzer

Deep URL inspection — checks for HTTP, URL shorteners, suspicious path keywords, and blacklisted domains.

📝

NLP Engine

Pattern-matching against 20+ spam & phishing signals. Distinguishes between phishing, promotional, and safe content.

🏷️

Brand Checker

Identifies trusted brands in email content and cross-checks domain alignment to catch impersonation attacks.

📡

Source Verification

Validates sender domains against trusted registries and flags mismatches between claimed and actual origins.

⚡

Weighted Scoring

Each module outputs a normalized score. A dynamic weight engine fuses them into a single explainable risk score.

Five Layers of
Threat Detection

Every email you paste runs through five independent analysis modules. Their scores are fused by a weighted engine to give you a final, explainable verdict.

01

Input & Preprocessing

The raw email text is lowercased, tokenized, and passed to all five modules simultaneously. No data is stored or transmitted to any server.

email = request.form["email"].lower()
02

NLP Text Analysis

Scans for 20+ spam/phishing patterns ("urgent", "verify OTP", "account blocked"), promotional signals ("discount", "sale", "cashback"), and safe keywords ("meeting", "report"). Returns separate spam and promo scores.

spam_patterns = ["urgent","verify","otp","password",...]
promo_patterns = ["discount","sale","offer","deal",...]
03

Link & URL Inspection

Extracts all URLs using regex and scores each one: HTTP (no SSL) = +1.5, URL shorteners = +2, suspicious path keywords = +2.5, blacklisted domains = +3, risky TLDs (.xyz, .ru) = +2. Capped at 8.

links = re.findall(r'(https?://\S+)', text)
04

ML Classification

A TF-IDF + Logistic Regression pipeline trained on a merged dataset of 10,000+ emails predicts spam (1) or ham (0). ML prediction contributes ×1.5 weight to the final score and can force-flag as spam.

model = Pipeline([("tfidf", TfidfVectorizer()), ("clf", LogisticRegression())])
05

Weighted Score Fusion

All module outputs are fused: final = (source×0.8) + (text×0.5) + (link×1.2) + (brand×1.0) + (ml×1.5). If any single module triggers a hard threshold, spam is forced regardless of total score.

final = (s*0.8)+(t*0.5)+(l*1.2)+(b*1.0)+(ml*1.5)+adjust
end
